# Casos de Uso

## Super Admin

- Realizar login.
- Cadastrar curso.
- Cadastrar coordenador.
- Vincular coordenador a cursos.
- Cadastrar aluno quando necessario.
- Criar regras de atividades complementares.
- Acompanhar dashboards globais.
- Revisar certificados.
- Consultar logs de auditoria.
- Acompanhar e-mails simulados.

## Coordenador

- Realizar login.
- Visualizar apenas cursos vinculados.
- Cadastrar alunos nos seus cursos.
- Publicar atividades para seus cursos.
- Criar regras dos seus cursos.
- Acompanhar envios de atividades.
- Aprovar ou reprovar submissões.
- Revisar certificados de alunos dos seus cursos.
- Processar OCR quando habilitado.

## Aluno web

- Realizar login.
- Visualizar progresso de carga horaria.
- Enviar comprovante de atividade.
- Enviar certificado em PDF ou imagem.
- Acompanhar status de validacao.
- Receber notificacoes internas.
- Participar de oportunidades.
